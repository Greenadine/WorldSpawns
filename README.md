# WorldSpawns
WorldSpawns is a Minecraft plugin that enables a server to have a unique spawn point set for each world on the server. Besides that, it also has a hub feature and more.

# Features
* Set a unique spawn for each different world.
* Set a hub for the server.
* Set a special spawnpoint for new players, which can be custom set, or just set to the hub.
* Create special signs for teleporting to a world's spawn or the hub.
* Make portals for teleporting to a world's spawn (W.I.P.).
